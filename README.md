# Grouping_In_Insurance
Code for publication on https://arxiv.org/abs/1912.09964 .

The provided files show the data for both term life insurance and pension plans. For step 1 (see paper), we show the uniformly generated data which is used for fitting the prediction model. Data for step 2 represents the realistic portfolio, which we eventually apply the novel grouping procedure to (given the neural networks fitted in step 1).
